TESTING CHECKLIST
Local Password Vault - Complete Testing Guide
================================================================================

This checklist covers all testing scenarios to verify the application works
correctly before release.

================================================================================
PRE-TESTING SETUP
================================================================================

Before starting tests, verify:
[ ] Backend server is accessible (https://api.localpasswordvault.com/health)
[ ] Backend server is running (pm2 status shows lpv-api as online)
[ ] Environment variables are set correctly on server
[ ] Stripe webhook is configured in Stripe Dashboard
[ ] Brevo email service is configured
[ ] Test license keys are available (if testing activation)
[ ] Test trial keys are available (if testing trial flow)

================================================================================
1. CONNECTION ERROR FIX VERIFICATION
================================================================================

Test the connection error fix:
[ ] Build the application with latest fixes
[ ] Install on clean Windows machine
[ ] Try to activate a license key
[ ] Verify no "Unable to connect to license server" error appears
[ ] If error appears, note the exact error message
[ ] Check electron log file for detailed error information
[ ] Test with network diagnostic feature (if implemented in UI)

Expected Results:
- Activation should succeed if backend is accessible
- Error messages should be specific and actionable
- Logs should contain detailed error information
