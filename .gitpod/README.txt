# Local Password Vault - GitPod Development Environment

This GitPod workspace is configured for secure vendor access to the Local Password Vault codebase.

## Security Features

- Code remains within the GitPod environment
- Access controls prevent unauthorized downloads
- All actions are logged and monitored
- Environment is ephemeral and destroyed after use

## Getting Started

1. The environment automatically installs dependencies when created
2. The development server starts automatically
3. For Electron development, open a terminal and run `npm run electron-dev`
4. Test the application in the preview window

## Environment Configuration

- Node.js 18
- Vite development server
- ESLint and Prettier for code quality
- TypeScript support
- Tailwind CSS tools

## Access Restrictions

- Repository cloning is disabled
- File downloads are restricted
- Network access is limited to required development services

For any issues or questions, please contact the project administrator.
