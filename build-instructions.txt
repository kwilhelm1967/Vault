# Building Local Password Vault

## Prerequisites

1. **Node.js** (v18 or higher)
2. **npm** or **yarn**

## Build Instructions

### 1. Install Dependencies
```bash
npm install
```

### 2. Build for Development
```bash
# Run in development mode (web + electron)
npm run electron-dev
```

### 3. Build for Production

#### Windows (.exe)
```bash
npm run dist
```
This creates:
- `release/LocalPasswordVault Setup.exe` (installer)
- `release/LocalPasswordVault-Portable.exe` (portable version)

#### macOS (.dmg)
```bash
npm run dist
```
Creates: `release/LocalPasswordVault.dmg`

#### Linux (.AppImage)
```bash
npm run dist
```
Creates: `release/LocalPasswordVault.AppImage`

- **Windows**: `LocalPasswordVault Setup.exe` and `LocalPasswordVault-Portable.exe`
- **macOS**: `LocalPasswordVault.dmg`
- **Linux**: `LocalPasswordVault.AppImage`

## Features

✅ **Truly Offline**: No internet connection required or used
✅ **Desktop Native**: Runs as a native desktop application
✅ **Portable**: Portable .exe version available for Windows
✅ **Secure**: AES-256 encryption with local storage only
✅ **Cross-Platform**: Windows, macOS, and Linux support
✅ **Auto-Lock**: Automatic locking after inactivity
✅ **Floating Panel**: Compact, draggable interface
✅ **Full Vault View**: Complete password management interface

## Security Features

- Master password protection
- AES-256 encryption
- Local storage only (no cloud)
- Auto-lock after 15 minutes
- Secure memory handling
- No telemetry or tracking

## Distribution

The portable .exe file can be distributed as a single file that users can run directly without installation.
