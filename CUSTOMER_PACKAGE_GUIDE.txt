## Creating Customer Packages

### Automated Creation

Use the provided scripts to automatically create all packages:

**Linux/macOS:**
```bash
chmod +x customer-packages/create-packages.sh
./customer-packages/create-packages.sh
```

**Windows:**
```cmd
customer-packages\create-packages.bat
```

### Manual Creation

If you need to create packages manually:

1. **Create Base Directory**
   ```bash
   mkdir -p dist-packages/single-user
   ```

2. **Copy Tier-Specific Files**
   ```bash
   cp -r customer-packages/single-user/* dist-packages/single-user/
   ```

3. **Copy Source Code**
   ```bash
   cp -r src dist-packages/single-user/
   cp -r public dist-packages/single-user/
   cp index.html dist-packages/single-user/
   ```

4. **Copy Configuration Files**
   ```bash
   cp package.json dist-packages/single-user/
   cp vite.config.ts dist-packages/single-user/
   cp tailwind.config.js dist-packages/single-user/
   ```

5. **Create ZIP Archive**
   ```bash
   cd dist-packages/single-user
   zip -r ../LocalPasswordVault-single-user.zip .
   ```

## Package Customization

### White-Label Customization

For partners who want to rebrand:

1. **Update Branding**
   ```bash
   # Replace logo files
   cp partner-logo.svg public/vault-icon.svg
   
   # Update company name in source
   find src -name "*.tsx" -exec sed -i 's/Local Password Vault/Partner Name/g' {} \;
   ```

2. **Customize Documentation**
   ```bash
   # Update README files
   sed -i 's/LocalPasswordVault.com/partner-domain.com/g' customer-packages/*/README.txt
   ```

3. **Configure Support Contacts**
   ```bash
   # Update support emails
   sed -i 's/support@LocalPasswordVault.com/support@partner.com/g' customer-packages/*/README.txt
   ```

### Custom Feature Sets

To create custom packages with specific features:

1. **Modify Feature Flags**
   ```typescript
   // src/config/environment.ts
   export const features = {
     enableCloudSync: false, // Disable for offline-only version
     maxDevicesPerLicense: {
       single: 1,
       family: 3,
       business: 10
     }
   };
   ```

2. **Update License Validation**
   ```typescript
   // src/utils/licensing.ts
   getFeatureLimits(): FeatureLimits {
     // Customize based on package tier
   }
   ```
