Local Password Vault - Installation Package
===========================================

Thank you for purchasing Local Password Vault!

This package contains:
- Installer file for your platform
- Documentation files
- Quick start guide

INSTALLATION INSTRUCTIONS
-------------------------

Windows:
1. Double-click the .exe installer file
2. Follow the installation wizard
3. Launch Local Password Vault from your Start menu

macOS:
1. Open the .dmg file
2. Drag Local Password Vault to your Applications folder
3. Launch from Applications

Linux:
1. Make the .AppImage file executable:
   chmod +x "Local Password Vault-*.AppImage"
2. Double-click to run, or run from terminal

ACTIVATION
----------

After installation:
1. Launch Local Password Vault
2. Enter your license key (sent to your email)
3. Your vault will be ready to use!

IMPORTANT: 100% OFFLINE OPERATION
----------------------------------

After activation, Local Password Vault works completely offline:
- No internet connection required
- No data sent to servers
- All encryption happens locally on your device
- Your passwords never leave your computer
