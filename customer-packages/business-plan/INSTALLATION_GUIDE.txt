## Troubleshooting Enterprise Deployment

### Common Deployment Issues

**MSI installation fails:**
```
Check Windows version compatibility
Run as administrator
Verify no conflicting software
Check Windows Installer service
```

**PKG installation blocked on macOS:**
```
System Preferences â†’ Security & Privacy
Click "Allow" for blocked installation
Or use: sudo installer -pkg file.pkg -target /
```

**License activation fails:**
```
Verify internet connectivity
Check firewall settings
Confirm license key accuracy
Contact enterprise support
```

### Network and Security Issues

**Sharing not working between devices:**
```
Check firewall rules (ports 8080-8090)
Verify network connectivity
Test with Windows Firewall disabled
Configure enterprise firewall exceptions
```

**Admin dashboard inaccessible:**
```
Verify admin license activation
Check user permissions
Restart application as administrator
Clear application cache
```

### Performance Optimization

**Large organization performance:**

Implement team-based organization

Monitor system resources

**Network sharing optimization:**
