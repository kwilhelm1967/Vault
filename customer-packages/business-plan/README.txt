## Quick Start Guide

### Step 1: Download Enterprise Packages

**Individual Installers:**
- Windows: LocalPasswordVault-Setup.exe
- macOS: LocalPasswordVault.dmg
- Linux: LocalPasswordVault.AppImage

**Enterprise Deployment Packages:**
- Windows MSI: LocalPasswordVault-Enterprise.msi
- macOS PKG: LocalPasswordVault-Enterprise.pkg
- Linux DEB: LocalPasswordVault-Enterprise.deb

### Step 2: Set Up Administrator Account

1. Install on your administrator device first
2. Launch Local Password Vault
3. Enter your Administrator license key
4. Accept the End User License Agreement
5. Create your admin master password
6. Access Settings > Business Administration
7. Set up your organization profile

### Step 3: Deploy to Your Team

**Option A: Individual Installation**
1. Send user license keys to team members
2. Have them install and activate individually
3. Invite them to your organization

**Option B: Managed Deployment**
1. Use MSI/PKG installers with your IT tools
2. Configure automatic license activation
3. Deploy organization-wide settings

### Step 4: Configure Team Management

1. Go to Admin Dashboard > Team Management
2. Add team members with their email addresses
3. Assign license keys and roles
4. Create teams and password collections
5. Set up sharing permissions
