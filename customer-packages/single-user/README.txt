## Quick Start Guide

### Step 1: Install the Application

Choose your operating system:

**Windows:**
1. Download: LocalPasswordVault-Setup.exe
2. Run the installer as administrator
3. Follow the installation wizard

**macOS:**
1. Download: LocalPasswordVault.dmg
2. Open the .dmg file
3. Drag the app to Applications folder

**Linux:**
1. Download: LocalPasswordVault.AppImage
2. Make it executable: `chmod +x LocalPasswordVault.AppImage`
3. Run the application

### Step 2: Activate Your License

1. Launch Local Password Vault
2. Enter your license key when prompted
3. Accept the End User License Agreement
4. Create your master password (IMPORTANT: Remember this!)

### Step 3: Start Using Your Vault

1. Click "Add Account" to add your first password
2. Fill in the account details
3. Use the password generator for strong passwords
4. Organize with categories (Banking, Shopping, etc.)
