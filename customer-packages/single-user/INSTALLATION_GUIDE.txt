## Installation Steps

### Windows Installation

1. **Download the Installer**
   - File: `LocalPasswordVault-Setup.exe`
   - Size: ~150 MB

2. **Run the Installer**
   ```
   Right-click LocalPasswordVault-Setup.exe
   Select "Run as administrator"
   ```

3. **Installation Wizard**
   - Click "Next" to begin
   - Accept the license agreement
   - Choose installation directory (default recommended)
   - Select "Create desktop shortcut" if desired
   - Click "Install"

4. **Complete Installation**
   - Wait for installation to complete
   - Click "Finish"
   - Launch from desktop shortcut or Start menu

### macOS Installation

1. **Download the Disk Image**
   - File: `LocalPasswordVault.dmg`
   - Size: ~150 MB

2. **Mount and Install**
   ```
   Double-click LocalPasswordVault.dmg
   Drag "Local Password Vault" to Applications folder
   Eject the disk image
   ```

3. **First Launch**
   ```
   Open Applications folder
   Double-click "Local Password Vault"
   If prompted about unidentified developer:
     - Right-click the app
     - Select "Open"
     - Click "Open" in the dialog
   ```

### Linux Installation

1. **Download the AppImage**
   - File: `LocalPasswordVault.AppImage`
   - Size: ~150 MB

2. **Make Executable**
   ```bash
   chmod +x LocalPasswordVault.AppImage
   ```

3. **Run the Application**
   ```bash
   ./LocalPasswordVault.AppImage
   ```

4. **Optional: Desktop Integration**
   ```bash
   # Create desktop entry
   mkdir -p ~/.local/share/applications
   cat > ~/.local/share/applications/local-password-vault.desktop << EOF
   [Desktop Entry]
   Name=Local Password Vault
   Exec=/path/to/LocalPasswordVault.AppImage
   Icon=password-manager
   Type=Application
   Categories=Utility;Security;
   EOF
   ```

## First-Time Setup

### 1. License Activation

When you first launch the application:

1. **Enter License Key**
   - You'll see the license activation screen
   - Enter your license key: `XXXX-XXXX-XXXX-XXXX`
   - Click "Activate License"

2. **Accept EULA**
   - Read the End User License Agreement
   - Scroll to the bottom
   - Check "I agree to the terms"
   - Click "Accept"

### 2. Create Master Password

1. **Choose a Strong Password**
   - Minimum 8 characters
   - Include uppercase, lowercase, numbers, symbols
   - Make it memorable but unique
   - **CRITICAL:** Write it down securely - we cannot recover it!

2. **Confirm Password**
   - Re-enter your master password
   - Ensure both entries match

3. **Set Security Questions**
   - Choose 3 security questions
   - Provide answers you'll remember
   - These help recover access if you forget your master password

### 3. Initial Configuration

1. **Import Existing Passwords (Optional)**
   - Click "Import" if you have passwords from another manager
   - Supported formats: CSV, JSON
   - Follow the import wizard

2. **Add Your First Password**
   - Click "Add Account"
   - Fill in account details
   - Use the password generator for strong passwords
   - Choose appropriate category

## Troubleshooting

### Windows Issues

**Installer won't run:**
- Run as administrator
- Temporarily disable antivirus
- Check Windows version compatibility

**Application won't start:**
- Install latest .NET Framework
- Run Windows Update
- Check for conflicting software

### macOS Issues

**"App can't be opened" error:**
- Right-click app â†’ Open
- Go to System Preferences â†’ Security & Privacy
- Click "Open Anyway"

**App crashes on startup:**
- Check macOS version compatibility
- Restart your Mac
- Try creating a new user account

### Linux Issues

**AppImage won't run:**
- Ensure execute permissions: `chmod +x LocalPasswordVault.AppImage`
- Install FUSE: `sudo apt install fuse` (Ubuntu/Debian)
- Try running from terminal to see error messages

**Missing dependencies:**
```bash
# Ubuntu/Debian
sudo apt install libgtk-3-0 libgconf-2-4 libnss3 libxss1

# CentOS/RHEL
sudo yum install gtk3 GConf2 nss libXScrnSaver

# Fedora
sudo dnf install gtk3 GConf2 nss libXScrnSaver
```
