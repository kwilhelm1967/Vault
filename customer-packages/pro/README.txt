## Quick Start Guide

### Step 1: Install the Application

Choose your operating system for each device:

**Windows:**
1. Download: LocalPasswordVault-Setup.exe
2. Run the installer as administrator
3. Follow the installation wizard

**macOS:**
1. Download: LocalPasswordVault.dmg
2. Open the .dmg file
3. Drag the app to Applications folder

**Linux:**
1. Download: LocalPasswordVault.AppImage
2. Make it executable: `chmod +x LocalPasswordVault.AppImage`
3. Run the application

### Step 2: Activate Your License

**Important:** You have 6 license keys - use a different key for each device.

1. Launch Local Password Vault on first device
2. Enter License Key #1 when prompted
3. Accept the End User License Agreement
4. Create your master password
5. Set up security questions

Repeat for devices 2-6 using their respective license keys.

### Step 3: Multi-Device Setup

1. Install on each device (up to 6 total)
2. Use the same license key for activation
3. Create unique master passwords for each device (recommended)
4. Import/export data between devices as needed
