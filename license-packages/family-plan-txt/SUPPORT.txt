## Common Issues & Solutions

### Installation & Activation

#### Multiple Device Activation
- Your Family Plan includes 3 license keys
- Use a different key for each device
- If you need more than 3 devices, consider upgrading to Business Plan

#### License Already Activated
- Each license key can only be used on one device
- If you're replacing a device, deactivate the old one first
- Contact priority support if you've lost access to the old device

### Cloud Sync Issues

#### Sync Not Working
- Verify internet connection on all devices
- Check that Cloud Sync is enabled in Settings
- Ensure you're using the same sync account on all devices
- Try signing out and back into your sync account

#### Sync Conflicts
- If multiple devices modified the same entry, a conflict may occur
- The app will prompt you to choose which version to keep
- You can also merge changes manually
- Regular syncing reduces the chance of conflicts

### Family Sharing

#### Can't See Shared Passwords
- Verify the password was shared with the correct email address
- Check that the recipient has accepted the sharing invitation
- Ensure both users have Cloud Sync enabled
- Try re-sharing the password

#### Sharing Permission Issues
- Verify the correct permission level was assigned (view-only or edit)
- The owner can modify permissions at any time
- Recipients cannot share passwords further unless given permission
- Contact priority support if permissions aren't working correctly

### Multi-Device Management

#### Device Limit Reached
- The Family Plan supports up to 3 devices
- Deactivate unused devices to free up licenses
- Consider upgrading to Business Plan for more devices

#### Deactivating Old Devices
1. On the old device, go to Settings > License
2. Click "Deactivate License"
3. Confirm deactivation
4. The license can now be used on another device

## Family-Specific Support

### Managing Family Access
- Each family member should have their own master password
- The account owner can manage sharing permissions
- Consider what passwords should be shared vs. private
- Set up appropriate categories for shared passwords

### Teaching Password Security
- Show family members how to use the password generator
- Explain the importance of unique passwords
- Demonstrate how to use the floating panel
- Set up regular security check-ins

### Child Safety Features
- Create appropriate categories for children
- Share only age-appropriate accounts
- Consider view-only permissions for younger users
- Use the activity logs to monitor usage

## How to Get Priority Support

### Before Contacting Support
1. Check the Knowledge Base for solutions
2. Search the Community Forum for similar issues
3. Try the troubleshooting steps in this guide
4. Restart the application and your devices

### When Contacting Priority Support
Please include:
- Your license key
- Device information (OS, version)
- Application version
- Detailed description of the issue
- Steps to reproduce the problem
- Any error messages (screenshots if possible)
- Which family members/devices are affected

### Priority Support Process
1. Email priority@LocalPasswordVault.com
2. Include all relevant information
3. Receive a priority support ticket number
4. Support will respond within 24 hours
5. Follow up on the same email thread
