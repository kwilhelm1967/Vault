## Common Issues & Solutions

### Installation & Activation

#### Multiple Device Activation
- Your Pro license includes 6 license keys
- Use the same key for each device (up to 6 devices)
- If you need more than 6 devices, consider upgrading to Business Plan

#### License Already Activated
- Each license key can be used on up to 6 devices
- If you're replacing a device, deactivate the old one first
- Contact priority support if you've lost access to the old device

### Multi-Device Management

#### Device Limit Reached
- The Pro Plan supports up to 6 devices
- Deactivate unused devices to free up licenses
- Consider upgrading to Business Plan for more devices

#### Deactivating Old Devices
1. On the old device, go to Settings > License
2. Click "Deactivate License"
3. Confirm deactivation
4. The license can now be used on another device

### Vault Access

#### Forgot Master Password
- Use your security questions if configured
- If security questions aren't set up, you'll need to reset your vault
- There is no way to recover your master password if security questions aren't set

#### Vault Won't Unlock
- Verify you're entering the correct master password
- Check your keyboard layout and caps lock
- Try restarting the application
- As a last resort, reset your vault

### Data Management

#### Import Failed
- Verify your import file format is supported
- Check that the file isn't corrupted
- Try exporting from the source in a different format
- Contact support with details about the source format

#### Export Failed
- Check you have write permissions to the destination
- Try a different location
- Restart the application and try again
- Contact support if the issue persists

## How to Get Priority Support

### Before Contacting Support
1. Check the Knowledge Base for solutions
2. Search the Community Forum for similar issues
3. Try the troubleshooting steps in this guide
4. Restart the application and your computer

### When Contacting Support
Please include:
- Your license key
- Operating system version
- Application version
- Detailed description of the issue
- Steps to reproduce the problem
- Any error messages (screenshots if possible)

### Support Request Process
1. Email priority@LocalPasswordVault.com
2. Include all relevant information
3. Receive a support ticket number
4. Support will respond within 24 hours
5. Follow up on the same email thread
